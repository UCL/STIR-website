Summary

STIR is Open Source software for use in tomographic imaging. Its aim is to provide a Multi-Platform Object-Oriented framework for all data manipulations in tomographic imaging. Currently, the emphasis is on image reconstruction in emission tomography (PET and SPECT) but other imaging modalities can be added in the future.

During this annual meeting experienced users and developers present their recent work with STIR with the emphasis on software and algorithmic development.

For up-to-date information on this meeting please always check the STIR website.

Abstract submission

If you are interested in presenting your recent work using and/or extending the STIR library, please submit an abstract via the official conference website. Although the abstract submission is closed for the main conference, it is still open for the workshops including our meeting. If your abstract is accepted, it will feature as part of the IEEE conference in the official programme of the meeting.
