Summary

STIR is Open Source software for use in tomographic imaging. Its aim is to provide a Multi-Platform Object-Oriented framework for all data manipulations in tomographic imaging. Currently, the emphasis is on image reconstruction in emission tomography (PET and SPECT) but other imaging modalities can be added in the future.

During this annual meeting experienced users and developers present their recent work with STIR with the emphasis on software and algorithmic development.

Ashley Gillman, Jye Smith, Jason Dowling, Paul Thomas, Stephen Rose, Nicholas Dowson
CSIRO Australian e-Health Research Centre and University of Queensland (Australia)
Title: A framework for generalised PET motion correction with STIR implemented in Nipype
Abstract: PET motion correction can be presented in a generalised framework, able to encapsulate a variety of PET motion correction techniques. Variations include tracking (respiratory bellows, MR navigator, optical, device-less tracking, etc.), discretisation (amplitude-based, phase-based, multiple frames, etc.), modelling (registration, optical, single-phase gating, etc.), and correction (MOLAR, RTA, MCIR). We present a Python implementation of the framework leveraging Nipype for workflow modelling. Nipype provides a low barrier to entry to using and producing reusable and composable workflows, and can interface with a wide variety of software, including registration packages such as Mirorr and ANTs. We also provide wrappers for many STIR tools, which can be used as a high-level Python interface to STIR, alongside the existing, lower-level Python SWIG interface. The framework is demonstrated on a Hoffman Phantom experiment using a PET-alone motion correction scheme, in which device-less techniques are used for tracking, and PET non-attenuation corrected (NAC) reconstruction are registered to derive motion models for use in a reconstruct-transform-add (RTA) correction scheme. Of the device-less techniques considered, sensitivity, centre-of-mass and principal component analysis (PCA), the latter is found to be superior for rigid motion.

Elise Emond, Institute of Nuclear Medicine, University College London (UK)
Nikos Efthimiou, University of Hull (UK)
Title: Time-of-Flight within STIR: overview and results of tests using GATE simulations
Abstract: Time-of-Flight functionality in STIR has been implemented and is currently in final testing phase. The presentation will focus on its specific workflow within STIR and how to unlist and reconstruct listmode data from GATE simulations and from some real scanner acquisitions, with and without Time-of-Flight. A brief explanation on the construction of a scanner in GATE and how to make it consistent with a scanner prototype in STIR will also be provided.
5.
Daniel Deidda, University of Leeds (UK)
Title: List-Mode Kernelized Expectation Maximization implementation in STIR
Abstract: Ordered subsets expectation maximization and Bayesian algorithms are implemented into the STIR library and many publication using these algorithm have been published. The recently introduced kernelized expectation maximization (KEM) method has shown promise for different applications. Therefore, a list-mode-hybrid KEM reconstruction algorithm was implemented. In this algorithm there is the option of choosing between a normal Kernel matrix and a hybrid kernel matrix. The proposed algorithm uses both the magnetic resonance image and PET previous estimate to create the kernel matrix.
6.
Kris Thielemans, Institute of Nuclear Medicine, University College London (UK)
Title: The CCP PET-MR Synergistic Image Reconstruction Framework (SIRF) and its relation to STIR
Abstract: The UK Collaborative Computational Project for Synergistic PET/MR Reconstruction (CCP PET-MR) has released its first version of SIRF, a new open source platform for dual-modality reconstruction. This talk will give an overview of the project and its software and how it relates to STIR.

Title: Current projects under development and plan for future releases.
Abstract: Various groups are working on extensions for STIR. I will give a brief summary of their aim and status. I will also present discuss future releases.
