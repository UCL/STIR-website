Detailed Description

A class for a general element (or "record") of a GE RDF9 listmode file.

All types of records are stored in a (private) union with the "basic" classes such as CListEventDataGEHDF5. This class essentially just forwards the work to the "basic" classes.

Takes the scanner and first_time stamp. The former will be used for checking and swapping, the latter for adjusting the time of each event, as GE listmode files do not start with time-stamp 0.

get_time_in_millisecs() should therefore be zero at the first time stamp.
