162 /* If you have an older compiler, chages are that std::copy is
163  implemented in the obvious way of iterating and copying along
164  the way. However, for simple types (such as floats), calling
165  memmove (not memcpy as the ranges could overlap) is faster.
166  So, we overload std::copy for some built-in types.
167 
168  However, newer compilers (in particular gcc from version 2.8)
169  take care of this themselves. So, we only do this
170  conditionally.
171 */
