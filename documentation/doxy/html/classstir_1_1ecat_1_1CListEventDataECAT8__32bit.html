Detailed Description

Class for decoding storing and using a raw coincidence event from a listmode file from the ECAT 966 scanner.

This class is based on Siemens information on the PETLINK protocol, available at http://usa.healthcare.siemens.com/siemens_hwem-hwem_ssxa_websites-context-root/wcm/idc/groups/public/@us/@imaging/@molecular/documents/download/mdax/mjky/~edisp/petlink_guideline_j1-00672485.pdf

This class just provides the bit-field definitions. You should normally use CListEventECAT8_32bit.

In the 32-bit event format, the listmode data just stores on offset into a (3D) sinogram. Its characteristics are given in the Interfile header.
