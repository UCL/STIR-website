Detailed Description

template<typename coordT>
class stir::CartesianCoordinate2D< coordT >

a templated class for 2-dimensional coordinates.

It is derived from Coordinate2D<coordT>. The only new methods are y(),x(), corresponding resp. to operator[](1), operator[](2)

Warning
The constructor uses the order CartesianCoordinate2D<coordT>(y,x)
