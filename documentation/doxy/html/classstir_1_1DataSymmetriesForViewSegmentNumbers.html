Detailed Description

A class for encoding/finding symmetries. Works only on ViewSegmentNumbers (instead of Bin).

This class (mainly used in RelatedViewgrams and the projectors) is useful to store and use all information on symmetries common between the image representation and the projection data.

The class mainly defines members to find basic ViewSegmentNumbers. These form a 'basis' for all ViewSegmentNumbers in the sense that all ViewSegmentNumbers can be obtained by using symmetry operations on the 'basic' ones.
