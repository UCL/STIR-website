◆ DICOM_datetime_to_secs_since_Unix_epoch()

double stir::DICOM_datetime_to_secs_since_Unix_epoch ( const std::string & str,
bool silent = false )

convert DICOM DT string to seconds since the Unix epoch (i.e. 1 Jan 1970 00:00:00 UTC)

  • str A DICOM date-time character string in the format: YYYYMMDDHHMMSS.FFFFFF&ZZXX
  • silent if true, silence warnings about TZ info not being present Time_Zone info is given by "&ZZXX", with & = "+" or "-", and ZZ = Hours and XX = Minutes of offset w.r.t. UTC. If no TZ is given, the local time_zone with DST is used. A warning() is then issued, unless silent=true.

References DICOM_datetime_to_secs_since_Unix_epoch(), error(), info(), standardise_interfile_keyword(), and time_zone_offset_in_secs().
