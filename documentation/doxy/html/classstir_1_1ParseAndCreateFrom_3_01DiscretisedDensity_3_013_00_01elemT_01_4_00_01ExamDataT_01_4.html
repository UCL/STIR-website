Detailed Description

template<class elemT, class ExamDataT>
class stir::ParseAndCreateFrom< DiscretisedDensity< 3, elemT >, ExamDataT >

parse keywords for creating a VoxelsOnCartesianGrid from ProjData etc

This specialisation adds keywords like size etc to the parser, and calls the VoxelsOnCartesianGrid constructor with ExamInfo and ProjDataInfo arguments to obtain an image that is suitable to store the reconstruction.

Assumes that ExamDataT has get_exam_info_sptr() and get_proj_data_info_sptr() members.

Todo
Currently only supports VoxelsOnCartesianGrid parameters (we could introduce another keyword to differentiate between types).
