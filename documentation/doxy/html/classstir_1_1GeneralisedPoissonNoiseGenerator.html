Detailed Description

Generates noise realisations according to Poisson statistics but allowing for scaling.

A scaling_factor is used to multiply the input data before generating the Poisson random number. This means that a scaling_factor larger than 1 will result in less noisy data.

If preserve_mean=false,, the mean of the output data will be equal to scaling_factor*mean_of_input, otherwise it will be equal to mean_of_input, but then the output is no longer Poisson distributed.
