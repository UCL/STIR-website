Detailed Description

Projection data info for data from a scanner with discrete dtectors organised by blocks.

This class also contains 2 deprecated functions specific for (static) full-ring PET scanners. In this case, it is assumed that for 'raw' data (i.e. no mashing) sinogram space is 'interleaved'. See documentation for ProjDataInfoCylindricalNoArcCorr.

Deprecated
This class will be removed in v7.0.
