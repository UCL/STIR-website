Detailed Description

Three-dimensional cuboid box.

Description
A point with coordinates coord is inside the shape if for x,y,z given by Shape3DWithOrientation::transform_to_shape_coords(coord)

Parameters
  Box3D Parameters:=
     length-x (in mm):= <float>
     length-y (in mm):= <float>
     length-z (in mm):= <float>
     ; any parameters of Shape3DWithOrientation
  End:=
