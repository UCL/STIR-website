Detailed Description

A general base class for timers. Interface is like a stop watch.

Example of usage:
DerivedFromTimer t;
t.start();
// do things
cerr << t.value();
// do things
t.stop();
// do things
t.reset()
// etc

Derived classes simply have to implement the virtual function double get_current_value()
