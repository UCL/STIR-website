Detailed Description

Perform timings.

Author
Kris Thielemans

This utility performs timings of various operations. This is mostly useful for developers, but you could use it to optimise the number of OpenMP threads to use for your data.

Run the utility without any arguments to get a help message. If you want to know what is actually timed, you will have to look at the source code.
