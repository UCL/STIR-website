Detailed Description

class for specifying a time interval via a dynamic scan

The dynamic scan can be either a sinogram of an image. It is read via DynamicProjData or DynamicDiscretisedDensity.

Example .par file
  ; example keyword in some .par file
   time interval type:= from dynamic data
     Absolute Time Interval from dynamic data :=
       filename:=scandata.S
       start time frame:=1
       end time frame:=1
     end Absolute Time Interval from dynamic data :=
