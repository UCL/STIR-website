Detailed Description

Class for decoding storing and using a raw coincidence event from a listmode file from the ECAT 966 scanner.

This class just provides the bit-field definitions. You should normally use CListEventECAT966.

For the 966 the event word is 32 bit. To save 1 bit in size, a 2d sinogram encoding is used (as opposed to a detector number on the ring for both events). Both bin and view use 9 bits, so their maximum range is 512 values, which is fine for the 966 (which needs only 288).
